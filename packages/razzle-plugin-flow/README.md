# razzle-plugin-flow

This package contains a plugin for using Flow with Razzle.

## Usage in Razzle Projects

Install the plugin:

```
yarn add razzle-plugin-flow
```

Initialize Flow:

```
yarn flow init
```

Using the plugin with the default options

```js
// razzle.config.js

module.exports = {
  plugins: ['flow'],
};
```
